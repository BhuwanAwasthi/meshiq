# Job Description — Senior Python Developer

**Job ID:** JOB-001
**Role title:** Senior Python Developer
**Department:** Engineering
**Match keywords:** python, backend, senior developer, API, microservices, AWS, software engineer

## Role summary
We are seeking a Senior Python Developer to design and build scalable backend
services. The role involves API development, system design, mentoring junior
engineers, and owning services end to end in production.

## Required qualifications
- 5+ years professional software development experience
- Strong Python proficiency (FastAPI or Django, async patterns)
- Experience designing and operating REST APIs at scale
- Solid understanding of relational databases and query optimization
- Cloud experience (AWS or Azure), CI/CD pipelines, containerization (Docker)
- Demonstrated system design ability for distributed systems

## Preferred qualifications
- Experience mentoring or leading a small team
- Message queues / event-driven architecture (Kafka, Service Bus)
- Observability tooling (logging, tracing, metrics)

## Evaluation rubric (for the interview evaluator)
Score each dimension 1–5:
1. Technical depth in Python and backend fundamentals
2. System design and architecture reasoning
3. Production ownership and debugging experience
4. Communication and ability to explain trade-offs
5. Leadership / mentorship signals
A strong hire scores 4+ on dimensions 1, 2, and 4.
