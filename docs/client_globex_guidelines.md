# Client QA Guidelines — Globex Financial

**Client ID:** CLIENT-GLOBEX
**Client name:** Globex Financial
**Service type:** Collections / accounts receivable (regulated)
**Match keywords:** Globex, Globex Financial, outstanding balance, payment plan, account GX-, collections

## Call handling standards (regulated — strict)
Globex is a financial services client. Collections calls are subject to
fair-practice regulations. Each rule is scored pass/fail; the call gets a
1–100 overall score. Regulated violations are weighted heavily.

### 1. Mandatory disclosure (regulated)
- Agent must state the call is "an attempt to collect a debt"
- Agent must state the call "may be recorded"
- Omitting either is a critical regulated violation

### 2. Identity and privacy (regulated)
- Agent must verify they are speaking with the account holder before
  disclosing any balance or account information
- Disclosing debt details to a third party is a severe violation

### 3. Prohibited conduct (regulated)
- No threats, intimidation, or harassment
- No calling outside permitted hours language
- No false statements about consequences

### 4. Professional handling
- Agent offers a payment plan option where appropriate
- Agent remains calm if the customer is upset
- Agent documents the agreed next step

### 5. Closing
- Agent summarizes any agreement reached
- Agent provides a reference number

## Scoring
- Regulated violations (disclosure, privacy, prohibited conduct) = critical (-30 each)
- Professional handling lapses = moderate (-10 each)
- Closing omissions = minor (-5 each)
- A passing call scores 85+ (higher bar due to regulation).
